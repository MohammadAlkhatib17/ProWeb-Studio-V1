## Data Flow

1. **User Activity** â†’ Triggers web vitals measurements
2. **web-vitals lib** â†’ Calls metric callbacks (onLCP, onCLS, onINP)
3. **useWebVitals** â†’ Processes metrics, adds context
4. **sendBeacon/fetch** â†’ POSTs to API endpoint
5. **API Route** â†’ Validates and stores event
6. **In-Memory Storage** â†’ Maintains recent events (max 1000)
7. **Dashboard** â†’ Fetches and displays events
8. **Auto-refresh** â†’ Polls API every 10s
